# 👥 Employee Attrition Analysis

## 📌 Project Overview
Analyzed 1,470+ employee records to identify 
key attrition factors using Python, SQL & Power BI.

## 🔄 Project Workflow
1. Data Cleaning - Python (Google Colab)
2. Data Storage & Querying - SQL
3. Dashboard & Visualization - Power BI

## 🛠️ Tools Used
- Python (Pandas, NumPy) - Data Cleaning
- SQL - Data Storage & Querying
- Power BI (DAX) - Dashboard
- Google Colab - Development Environment

## 📊 Key Insights
- Attrition Rate : 16.12%
- Total Employees : 1,470
- Active Employees : 1,233
- Avg Salary : 6.50K
- R&D Dept : Highest Attrition (133)
